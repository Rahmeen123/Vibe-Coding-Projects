# Instructor Resources

This directory contains official materials, examples, and resources provided by the course instructor.

## 📂 What's Inside?
- **Lectures**: Slide decks and presentation materials.
- **Reference Code**: Sample implementations of AI concepts discussed in class.
- **Assignments**: Detailed instructions and starter code for course projects.
- **Resource Links**: Curated lists of documentation, tutorials, and tools.

## 📖 Usage
- Students are encouraged to explore the reference code to understand best practices.
- Use the assignment templates provided here when starting new projects.

*Note: This directory is primarily managed by the instructor. Students should not modify files here unless explicitly instructed.*
