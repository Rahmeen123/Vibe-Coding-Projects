# AI Code Reviewer & Bug Explainer

This document describes the technical skills, tools, and technologies required to develop the AI Code Reviewer & Bug Explainer system.

## Programming Languages

The project requires knowledge of the following programming languages:

- Python
- JavaScript
- Java
- C++
- C

## Frontend Development Skills

Frontend technologies are used to create the user interface where users can input code and view AI-generated results.

Required skills include:

- HTML
- JavaScript
- Bootstrap
- Basic UI design

## Backend Development Skills

The backend is responsible for processing user requests and communicating with AI APIs.

Required backend skills include:

- Python
- FastAPI framework
- REST API development
- Request and response handling

## AI & Machine Learning Skills

The project uses Artificial Intelligence models to analyze programming code and provide suggestions.

Required AI-related skills include:

- Understanding of Large Language Models (LLMs)
- Prompt engineering
- AI API integration
- LangChain framework

## AI APIs

The project integrates external AI services for code analysis.

APIs used include:

- OpenAI API
- Claude API

## Development Tools

The following tools are used during development:

- Visual Studio Code
- Web browser
- Internet connection for API access

## Additional Skills

Other useful skills for this project include:

- Debugging
- Problem solving
- Basic software architecture understanding
- API integration
