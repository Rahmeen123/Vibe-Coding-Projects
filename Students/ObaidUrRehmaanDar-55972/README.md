# Mela – AI-Driven Event Ecosystem

**A Hyper-Local AI Event Discovery Platform for Lahore**

## 🌟 Overview

Mela is a centralized, AI-powered platform designed to bridge the "information gap" in Lahore's academic and tech landscape. It automates event discovery by scouting professional networks and using AI to extract data from event posters.

## 🚀 Key Features

- **AI Scout:** Automated crawling of LinkedIn and web platforms for tech events.
- **Visual Reader (OCR):** Powered by Gemini 1.5 Flash to convert flyers into structured data.
- **Lahore-Centric Feed:** A real-time dashboard for students across FAST, Riphah, LUMS, and more.
- **Smart Categorization:** Automatic tagging using AI (e.g., #AI, #UIUX, #Hackathon).

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS (Material 3 Aesthetic)
- **Backend:** Firebase (Firestore, Auth, Cloud Functions)
- **AI:** Gemini 1.5 Flash
- **Automation:** Node.js + Puppeteer
