# Students Directory

Welcome to the Students directory of the AI Driven Development Course! This is where all coursework, individual projects, and progress tracking files are stored.

## 📂 Structure
Each student has their own dedicated folder following the naming convention:
`FirstNameLastName-SapID`

## 📝 Guidelines for Students
1. **Personal Folder**: If you haven't already, create a folder named `FirstNameLastName-SapID`.
2. **Project Organization**: Inside your folder, create sub-directories for each project or assignment.
3. **Documentation**: Each project should have its own `README.md` explaining the purpose, implementation details, and how to run the code.
4. **Commits**: Use descriptive commit messages when pushing your work (e.g., `feat: Add sentiment analysis project`).

## 🚀 Getting Started
To add your work:
1. Fork the repository.
2. Clone your fork locally.
3. Create/update your student folder.
4. Push changes and open a Pull Request.

Happy Coding! 🤖
