# Collaborative Projects

This directory is the hub for team-based projects and shared initiatives within the AI Driven Development Course.

## 🤝 Collaboration Policy
1. **Team Spirit**: All projects here are collaborative. Respect your teammates and contribute constructively.
2. **Project Proposal**: Before starting a new collaborative project, raise an issue with the `project-proposal` tag.
3. **Organization**: Each project should have its own folder with a clear, descriptive name.
4. **Main README**: Each project folder **must** contain a `README.md` documenting the project goals, team members, and setup instructions.

## 🚀 How to Participate
- Browse existing projects and look for issues with the `collaboration-wanted` tag.
- Coordinate with team members via GitHub Issues and Discussions.
- Follow the branching and PR guidelines outlined in the root `CONTRIBUTING.md`.

Let's build something amazing together! 🌟
